# Zen Palette

> **A command palette you control.**

Zen Palette is an open-source extension for Firefox-based browsers that provides a modern, keyboard-first command palette with one simple goal:

**Give users control over how browser results are presented.**

Inspired by the user experience of browsers like Zen Browser, Zen Palette focuses on being fast, predictable and configurable instead of relying on opaque ranking heuristics.

---

## Why?

Modern browsers combine many different sources into a single search experience:

* Search suggestions
* Visited pages
* Bookmarks
* Open tabs
* URLs

Every browser decides for you which result should come first.

Sometimes that's exactly what you wanted.

Sometimes it isn't.

Zen Palette believes this decision should belong to the user.

---

## What makes Zen Palette different?

Instead of asking:

> *"What does the browser think I want?"*

Zen Palette asks:

> *"How do **you** want your command palette to behave?"*

Every workflow is different.

Your command palette should be too.

---

## Features

### 🎯 Keyboard First

Open the palette.

Start typing.

Navigate with the keyboard.

Press **Enter**.

That's it.

---

### ⚙️ Fully Configurable

Customize the behavior of the palette.

Examples include:

* Enable or disable providers
* Reorder providers
* Configure provider priorities
* Choose your preferred search engine
* Define default actions

Your browser.

Your workflow.

Your rules.

---

### 🧩 Provider-Based Design

Results come from independent providers.

Examples:

* 🔍 Web Search
* 🌐 URLs
* ⭐ Bookmarks
* 🕘 History
* 📑 Open Tabs

Each provider can be configured independently.

---

## Example

Searching for:

```text
git
```

One user might prefer:

```text
🔍 Search "git"

⭐ GitHub

🕘 History
```

Another user might prefer:

```text
⭐ GitHub

🔍 Search "git"

🕘 History
```

Zen Palette doesn't decide.

You do.

---

## Project Principles

Zen Palette is built around four principles.

### Predictable

The interface should behave consistently.

### Configurable

Users should control the ranking—not hidden heuristics.

### Keyboard First

Everything should be accessible from the keyboard.

### Minimal

The palette should stay focused, fast and distraction-free.
